## Summary
The project uses approximately 85+ different Godot classes, focusing heavily on:
- 3D physics (RigidBody3D, CollisionShape3D, joints)
- UI elements (Control, Label, Button, containers)
- Animation system (AnimationPlayer, AnimationLibrary)
- Procedural mesh generation (ArrayMesh, various mesh types)
- Particle effects (GPUParticles3D)
- Threading and async operations (Thread, Mutex)
- File I/O and configuration (FileAccess, ConfigFile)

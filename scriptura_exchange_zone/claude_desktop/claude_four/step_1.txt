# Menu System & Navigation Implementation

## Features from Recipe:
- First screen with 3D world background showing last played save
- Main Menu with buttons to MultiverseMenu, UniverseMenu, OptionsMenu, AboutPage
- Exit button functionality
- Back button system
- W/S keyboard navigation and mouse support
- Button selection with Enter key

## Implementation Steps:

### Step 1: Create Base Menu Scene
1. Create `BaseMenu.tscn` with:
   - Background viewport for 3D world
   - Container for buttons
   - Navigation controller script
2. Set up scene inheritance system

### Step 2: Implement MainMenu
1. Create `MainMenu.tscn` inheriting from BaseMenu
2. Add buttons:
   - ButtonMultiverseMenu
   - ButtonUniverseMenu
   - ButtonOptionsMenu
   - ButtonAboutPage
   - exitButton
   - languageButton
3. Implement button click handlers

### Step 3: Create Navigation System
1. Implement keyboard navigation (W/S keys)
2. Add mouse hover detection
3. Create selection highlighting system
4. Implement Enter key activation

### Step 4: Create Scene Manager
1. Create SceneManager singleton
2. Implement scene transition methods
3. Add scene history for back button
4. Handle cleanup between scenes

### Step 5: Implement 3D Background
1. Create 3D viewport node
2. Load last saved universe/multiverse
3. Fallback to default image if first launch
4. Implement background animation
